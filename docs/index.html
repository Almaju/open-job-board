Quick Start

Query jobs

# List recent remote jobs
curl "https://ppubgurkauoptjsuyfff.supabase.co/rest/v1/jobs?remote_full=eq.true&order=posted_at.desc&limit=10" \
  -H "apikey: sb_publishable_S9AJoMdDWsX1cPOXpM3LJw_pqdn9U0B"

# Full-text search
curl -X POST "https://ppubgurkauoptjsuyfff.supabase.co/rest/v1/rpc/search_jobs" \
  -H "apikey: sb_publishable_S9AJoMdDWsX1cPOXpM3LJw_pqdn9U0B" \
  -H "Content-Type: application/json" \
  -d '{"query": "senior golang engineer", "country": "Germany", "remote": true}'

Submit a job

curl -X POST "https://ppubgurkauoptjsuyfff.supabase.co/functions/v1/submit-job" \
  -H "Content-Type: application/json" \
  -d '{
    "origin": { "source": "my-scraper", "reference": "job-12345" },
    "title": "Senior Backend Engineer",
    "description": "We are looking for a backend engineer...",
    "company": { "name": "Acme Corp", "sector": "Technology" },
    "location": { "city": "Berlin", "country": "Germany", "remote": { "full": true } },
    "employment_type": "full-time",
    "salary": { "currency": "EUR", "min": 80000, "max": 120000, "period": "yearly" }
  }'

How it works

Open Submissions

Submit jobs via a simple REST API. No account required. Rate limited to 10 req/min per IP. Use an API key for higher limits.

Deduplication

Scrapers can re-submit safely. The API returns HTTP 409 when the same source + reference already exists â€” no duplicates.

Full-Text Search

Search across title, company, description, and location with relevance ranking via PostgreSQL tsvector.

Powerful Filtering

Filter by country, city, remote, employment type, salary range, and source. Sort by recency or relevance.

Get an API Key

For trusted scrapers or high-volume use cases, open a GitHub issue with your scraper name, estimated volume, and contact email. API keys unlock higher rate limits.
